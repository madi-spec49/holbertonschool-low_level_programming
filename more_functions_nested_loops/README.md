# 📘 README – Fonctions et Boucles en C

## 🧠 Introduction

En langage C, les **fonctions** et les **boucles** sont des éléments fondamentaux pour structurer un programme et éviter la répétition de code.

---

## 🔹 Les fonctions

### 📌 Définition

Une fonction est un bloc de code réutilisable qui effectue une tâche spécifique.

---

### 🧱 Syntaxe

```c
type_retour nom_fonction(paramètres)
{
    // instructions
    return valeur;
}
